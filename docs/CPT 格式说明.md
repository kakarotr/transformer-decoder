# CPT 语料 Markdown 格式规范

## 间距规则

| 位置 | 间距 |
|------|------|
| 一级标题（`#`）与导言（仅维基百科） | 两个换行符 |
| 其他标题与所属段落 | 一个换行符 |
| 段落与段落 | 两个换行符 |
| 父标题无所属段落 → 子标题 | 两个换行符 |
| 列表项之间 | 一个换行符 |
| 列表与其他段落 | 两个换行符 |
| Topic Label（`**dt**`）与下方内容 | 一个换行符 |
| Topic Label 块与块之间 | 两个换行符 |

## 符号规则

| 元素 | 语法 |
|------|------|
| 一级标题（文章名/书名） | `#` |
| 二至四级标题 | `##`、`###`、`####` |
| 行内加粗 | `**text**` |
| Topic Label（`<dl>` 的 `<dt>`） | `**text**` 独占一行 |
| 无序列表 | `- ` |
| 有序列表 | `1. ` |
| 嵌套列表 | 缩进两个空格 |
| 引用（`<blockquote>`） | 不使用 `>` 语法，保留为普通段落，出处以 em-dash（——）内联追加 |

## 数据源格式差异

维基百科文章：`#` 标题下有导言段落，两个换行符分隔。

书籍：`#` 标题下无导言，直接接 `##` 章节标题，两个换行符分隔。

---

## 示例：维基百科文章

~~~markdown
# 织田信长

织田信长是日本战国时代至安土桃山时代的大名，出生于尾张国。他通过一系列军事与政治手段，结束了长期的群雄割据局面，为日本统一奠定了基础。信长以革新性的战术和对传统权威的挑战著称，被后世视为"天下布武"理念的实践者。

## 生涯

### 家督继承
天文二十年（1551年），织田信秀病逝，信长继承家督之位。由于信长年少时行为放荡不羁，被家臣称为"尾张的大傻瓜"，织田家内部对其能否胜任家督一职多有质疑。

信长的弟弟**织田信行**获得了部分家臣的支持，家督之争在此后数年间持续发酵。

### 桶狭间之战
永禄三年（1560年），今川义元率领约两万五千兵力西进，意图上洛。信长以不足三千的兵力，趁暴雨奇袭今川本阵，讨取义元。此战是信长从尾张一地方势力跃升为天下争夺者的转折点。

战后信长改变了尾张的外交方针，与三河的**德川家康**缔结同盟，即清洲同盟。这一同盟此后维持了近二十年，成为信长势力扩张的东方屏障。

## 政策

### 乐市乐座
信长在其领地内推行乐市乐座政策，废除既有的行会垄断特权，允许商人自由交易。

这一政策的目的在于促进城下町的商业繁荣，同时削弱旧有既得利益集团对经济活动的控制。

### 宗教政策
信长对佛教势力采取强硬态度，尤其是与一向一揆的长期对抗。

元龟二年（1571年），信长下令焚毁比叡山延历寺，此举震动天下。据路易斯·弗洛伊斯记载，信长认为延历寺的僧兵已成为阻碍统一的武装势力，而非单纯的宗教机构——《日本史》

## 人物·逸话

### 人物

**容貌**
信长身材高大挺拔，面容清秀。路易斯·弗洛伊斯描述其声音尖锐，好习武艺。

**性格**
信长性格果断刚烈，不拘泥于传统礼法。年少时因行为乖张被称为"大うつけ"，但其侧近多认为这并非愚钝，而是对既有秩序的蔑视。

### 逸话

**敦盛**
信长在桶狭间出阵前，舞了一段"敦盛"。其中的名句"人间五十年，与天相比，不过梦幻"常被后世引用，视为信长视死如归精神的象征。

## 家臣团

### 主要家臣

- **羽柴秀吉**
  原为低级武士，因才能被信长破格提拔。先后担任长浜城主、中国方面军司令官，负责对毛利氏的攻略。
- **明智光秀**
  出身美浓土岐氏支流，文武兼备。信长授予其近畿方面的军事指挥权，但最终于本能寺之变中叛变。
- **柴田胜家**
  信长早期的宿老，被委以北陆方面军的指挥。以勇猛著称，人称"�的破柴田"。

### 军团编制

1. 北陆方面军——柴田胜家率领，负责对上杉氏的攻略
2. 中国方面军——羽柴秀吉负责，主攻毛利氏
3. 关东方面军——的�的�的康政协助，牵制北条氏

### 家臣层级

- 宿老
  - 柴田胜家
  - 丹羽长秀
- 方面军司令
  - 羽柴秀吉
  - 明智光秀
~~~

## 示例：书籍

~~~markdown
# 信长的战争

## 序章
本书旨在从军事史的角度重新审视织田信长的统一事业。长期以来，信长的形象被"革新者"这一标签所固化，但若仔细考察其具体的战役指挥与兵站运营，可以发现其军事行动中既有突破常规的大胆决策，也有大量依循当时惯例的稳健操作。

## 第一章 尾张统一战
织田信长继承家督后，首先面对的并非外部强敌，而是族内的分裂。织田家分为多个支系，信长所在的弹正忠家仅为其中之一，且内部亦有信行一派的挑战。

信长对族内对手的处理方式表现出一个显著特征：在军事对决中毫不留情，但在事后的政治安排上往往保留对方势力的存续空间，以避免引发更大规模的连锁反抗。
~~~
